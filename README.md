## Pre-process
1. 卸载所有 OEM 预装软件。
2. 所有软件能从 Windows Store 安装就从这里安装。
3. 如果电脑有分区，把 文档， 视频， 图片等修改到其他地方。
4. 图吧工具箱: CPU-Z, GPU-Z, AIDA64, DisplayX
5. no@thanks.com
